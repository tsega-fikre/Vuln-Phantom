# Security Policy for VulnPhantom

## Responsible Use

VulnPhantom is a **web vulnerability scanner** designed strictly for **ethical hacking and penetration testing on systems you own or have explicit permission to test**.  

**Do NOT use VulnPhantom for illegal activities**, unauthorized hacking, or attacking systems without consent. Doing so is illegal and can result in criminal charges.  

## Reporting Vulnerabilities

If you discover a security vulnerability in VulnPhantom itself:  
- Please report it via email to: **nilephantom2@gmail.com**  
- Do not publicly disclose it until a fix is released.
